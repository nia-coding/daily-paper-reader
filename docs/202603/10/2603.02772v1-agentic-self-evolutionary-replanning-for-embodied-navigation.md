## 摘要
在复杂环境下的具身导航中，失败是不可避免的。为了增强韧性，重规划（RP）是一个可行的选择，它允许机器人失败，但能够调整计划直到成功。然而，现有的重规划方法冻结了自身动作模型，错失了通过升级机器人自身来探索更好计划的机会。为了解决这一局限性，我们提出了自进化重规划（Self-Evolutionary RePlanning，简称 SERP），通过从近期经验中进行运行时学习，实现了从冻结模型向进化模型的范式转变。与通常受限于预定义静态参数的现有模型进化方法不同，我们引入了智能体自进化动作模型，该模型利用带有自动微分的上下文学习（ILAD）进行自适应函数调整和全局参数重置。为了实现 SERP 的高效 Token 重规划，我们还提出了图思维链（GCOT）重规划，利用大语言模型（LLM）在蒸馏图上进行推理。广泛的仿真和真实世界实验表明，SERP 在各种基准测试中以更低的 Token 消耗实现了更高的成功率，验证了其在不同环境下的卓越鲁棒性和效率。

## Abstract
Failure is inevitable for embodied navigation in complex environments. To enhance the resilience, replanning (RP) is a viable option, where the robot is allowed to fail, but is capable of adjusting plan until success. However, existing RP approaches freeze the ego action model and miss the opportunities to explore better plans by upgrading the robot itself. To address this limitation, we propose Self-Evolutionary RePlanning, or SERP for short, which leads to a paradigm shift from frozen models towards evolving models by run-time learning from recent experiences. In contrast to existing model evolution approaches that often get stuck at predefined static parameters, we introduce agentic self-evolving action model that uses in-context learning with auto-differentiation (ILAD) for adaptive function adjustment and global parameter reset. To achieve token-efficient replanning for SERP, we also propose graph chain-of-thought (GCOT) replanning with large language model (LLM) inference over distilled graphs. Extensive simulation and real-world experiments demonstrate that SERP achieves higher success rate with lower token expenditure over various benchmarks, validating its superior robustness and efficiency across diverse environments.

### 1. 核心问题与研究动机
具身导航（Embodied Navigation）旨在将自然语言指令转化为机器人的可执行动作。尽管现有的重规划（Replanning, RP）方法在处理语义失败（全局层面）或物理失败（局部层面）方面取得了一定进展，但仍存在两大核心局限：
*   **模型冻结（Frozen Models）：** 现有的重规划方法在运行期间保持机器人自身的动作模型参数不变。当遇到环境不确定性或自身模型偏差时，机器人往往只能“绕过”失败（如寻找更远的路径），而无法通过“升级”自身参数来“克服”失败，导致效率低下或重复犯错。
*   **推理效率低下：** 在大型语义空间中进行迭代重规划会导致极高的 Token 消耗和计算延迟。

### 2. 方法论
论文提出了 **SERP（Self-Evolutionary RePlanning）** 框架，核心包含两个协同组件：

*   **局部智能体自进化（Local Agentic Self-Evolution, ASE）：**
    *   **核心思想：** 通过运行时学习更新机器人的运动规划参数（如路径跟踪权重、避障权重等）。
    *   **关键技术（ILAD）：** 结合了**带自动微分的上下文学习（In-context Learning with Auto-Differentiation）**。
        *   **AD（自动微分）：** 通过反向传播优化闭环损失函数，微调控制参数。
        *   **IL（上下文学习）：** 当 AD 陷入局部最优时，利用视觉语言模型（VLM）分析失败上下文，进行全局参数重置或调整损失函数权重，实现“跳出局部最优”。
    *   **参数检索（RAG）：** 利用检索增强生成技术，从记忆数据库中提取与当前场景匹配的初始参数。

*   **全局图思维链（Global Graph Chain-of-Thought, GCOT）：**
    *   **核心思想：** 将庞大的原始场景图蒸馏为更小、语义相关的任务子图。
    *   **流程：** 首先利用 LLM 将任务分解为子任务；接着使用 CLIP 模型（而非 LLM）进行特征匹配，快速检索候选元素；最后由 LLM 基于这些精简元素构建任务特定图并生成计划。这种“搜索与推理分离”的策略显著降低了 Token 消耗。

*   **实验规模：** 进行了 540 次仿真试验（180 任务 × 3 试验），并开展了真实世界机器人（配备 3D 激光雷达、RGBD 相机、Intel NUC）的验证。

### 6. 主要结论与发现
*   **性能提升：** SERP 在规划任务中的成功率比 SayPlan 高出 10% 以上。
*   **效率极高：** 在搜索和规划任务中，SERP 的 Token 节省率（RGTR）分别达到了 97% 和 93.9%，远优于 SayPlan。
*   **自进化优势：** 相比于纯梯度下降（AD），SERP 的 ILAD 机制在参数偏差较大时（60%-90% 偏差），成功率高出 AD 约 42%，且适应轮数更少。
*   **真实世界适应性：** 机器人能够通过实时调整控制参数（如降低路径跟踪权重以绕过窄门障碍），完成原本会失败的任务。

### 7. 优点
*   **范式创新：** 提出了从“冻结模型”到“自进化模型”的转变，使机器人具备了运行时自我修正的能力。
*   **协同优化：** 巧妙结合了 LLM 的推理能力（IL）和数值优化的精确性（AD），解决了复杂非凸空间下的参数调优问题。
*   **Token 经济性：** 通过 CLIP 预筛选和图蒸馏，极大地降低了使用大模型的成本和延迟。

### 8. 不足与局限
*   **短时记忆风险：** GCOT 采用的精简图策略虽然节省了 Token，但在极少数情况下可能会漏掉关键的搜索目标（如在 00861 场景中表现略逊于全图方法）。
*   **模型依赖：** 系统的性能高度依赖于底层 LLM（如 GPT-4o）的逻辑推理能力，使用较弱的模型可能会导致成功率下降。
*   **实时性挑战：** 尽管单次迭代较快，但在极度复杂的动态环境中，频繁调用大模型进行重规划仍可能产生感知与动作之间的滞后。
