## 摘要
准确感知物体硬度对于安全且灵巧的富接触机器人操作至关重要。在此，我们提出了 TactEx，这是一个统一了视觉、触觉和语言的可解释多模态机器人交互框架，用于类人硬度估计和交互式引导。我们在水果成熟度评估这一需要触觉感知和上下文理解的代表性任务上对 TactEx 进行了评估。该系统融合了 GelSight-Mini 触觉流、RGB 观测和语言提示。ResNet50+LSTM 模型从序列触觉数据中估计硬度，而跨模态对齐模块则将视觉线索与大语言模型（LLM）的引导相结合。这种可解释的多模态界面允许用户区分成熟度等级，并具有显著的统计学类别分离度（所有水果对的 p < 0.01）。在触碰位置选择方面，我们对比了 YOLO 与 Grounded-SAM (GSAM)，发现 GSAM 在细粒度分割和接触点选择方面更为鲁棒。一个轻量级 LLM 解析用户指令，并生成与触觉输出相关联的具身自然语言解释。在端到端评估中，TactEx 在简单用户查询上达到了 90% 的任务成功率，并且无需大规模微调即可泛化到新任务。这些结果突显了将预训练视觉和触觉模型与语言对齐相结合的潜力，以推进机器人领域中可解释的、类人的触觉感知和决策。

## Abstract
Accurate perception of object hardness is essential for safe and dexterous contact-rich robotic manipulation. Here, we present TactEx, an explainable multimodal robotic interaction framework that unifies vision, touch, and language for human-like hardness estimation and interactive guidance. We evaluate TactEx on fruit-ripeness assessment, a representative task that requires both tactile sensing and contextual understanding. The system fuses GelSight-Mini tactile streams with RGB observations and language prompts. A ResNet50+LSTM model estimates hardness from sequential tactile data, while a cross-modal alignment module combines visual cues with guidance from a large language model (LLM). This explainable multimodal interface allows users to distinguish ripeness levels with statistically significant class separation (p < 0.01 for all fruit pairs). For touch placement, we compare YOLO with Grounded-SAM (GSAM) and find GSAM to be more robust for fine-grained segmentation and contact-site selection. A lightweight LLM parses user instructions and produces grounded natural-language explanations linked to the tactile outputs. In end-to-end evaluations, TactEx attains 90% task success on simple user queries and generalises to novel tasks without large-scale tuning. These results highlight the promise of combining pretrained visual and tactile models with language grounding to advance explainable, human-like touch perception and decision-making in robotics.

#### 1. 核心问题与整体含义（研究动机和背景）
在机器人灵巧操作中，准确感知物体的硬度（Hardness）对于安全抓取和精细处理至关重要。人类能够通过触摸轻松判断物体的物理属性（如水果是否成熟），并能给出合理解释。然而，现有的机器人系统在硬度感知上面临三大挑战：
*   **细粒度区分难**：传统的分类模型难以区分硬度接近的物体（如不同成熟阶段的水果）。
*   **缺乏多模态融合**：视觉往往无法准确反映物体的内在机械属性，而纯触觉系统缺乏上下文理解。
*   **不可解释性**：大多数系统仅输出数值或标签，无法向人类用户解释其推理逻辑和不确定性。
**TactEx** 旨在通过整合视觉、触觉和语言模型，构建一个类人的、可解释的交互框架，以解决上述问题。

#### 2. 方法论：核心思想与关键技术
TactEx 采用模块化架构，分为三个主要阶段：
*   **视觉定位与目标对齐**：
    *   利用 **Grounded-SAM (GSAM)** 进行零样本（Zero-shot）目标识别。根据用户的自然语言指令（如“哪个水果最软？”），系统生成分割掩码并计算物体的 3D 质心，引导机器人进行触觉探测。
*   **触觉感知与硬度回归**：
    *   **硬件**：使用高分辨率的 **GelSight-Mini** 触觉传感器。
    *   **模型架构**：采用 **ResNet50 + LSTM** 的组合。ResNet50 提取单帧触觉图像特征，LSTM 处理探测过程中的时序序列（通常为 2 或 4 帧）。
    *   **损失函数**：为了防止模型坍缩（即所有预测值趋向均值），引入了自定义损失函数：
        $$L = MSE(p, l) + \lambda \cdot \min\left(\frac{1}{Var(p) + \epsilon}, 1000\right)$$
        该公式在最小化均方误差的同时，惩罚预测值的低方差，强制模型捕捉细微差异。
*   **语言生成与解释**：
    *   使用 **DeepSeek-R1-Distill-Llama-70B** 模型。它接收物体名称、空间位置和估计的硬度值，根据预设的成熟度阈值和规则，生成自然语言解释（如“左侧的香蕉较硬，尚未成熟”）。

#### 6. 主要结论与发现
*   **模型性能**：ResNet50-LSTM3 在硬度回归任务中表现最优（RMSE 4.30, $\rho$=0.88），显著优于 VGG 架构。
*   **视觉鲁棒性**：虽然 YOLO 的置信度更高，但 **GSAM 在分割精度（IoU）和质心定位误差上表现更好**，更适合引导触觉传感器进行精准接触。
*   **统计显著性**：TactEx 能够以极高的统计显著性区分所有测试水果的成熟度等级。
*   **端到端表现**：在简单查询下达到 90% 的成功率，但在处理未明确提及名称的复杂场景（如“总结桌上所有水果的硬度”）时，成功率会有所下降。

#### 7. 优点
*   **多模态闭环**：成功将视觉定位、触觉测量与 LLM 的逻辑推理整合进一个统一的交互框架。
*   **数据效率高**：通过预训练+微调策略，仅需 280 个样本即可在特定任务上实现高精度回归。
*   **可解释性强**：LLM 不仅给出数值，还能结合空间上下文和领域知识（如成熟度标准）提供人类可理解的理由。

#### 8. 不足与局限
*   **实时性欠佳**：由于采用串行处理流程（视觉->动作->触觉->语言），系统存在明显的延迟（Latency），无法实现实时的闭环调整。
*   **类别偏差**：实验发现模型对某些特定水果（如奇异果）的识别率较低，反映出视觉训练数据分布不均的问题。
*   **应用限制**：目前的硬度估计仍依赖于垂直向下的受控探测动作，对于复杂几何形状或动态环境下的硬度感知尚待验证。
