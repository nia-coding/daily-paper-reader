## 摘要
探索仍然是使用强化学习训练的大语言模型智能体的关键瓶颈。虽然先前的方法利用了预训练知识，但它们在需要发现新状态的环境中表现不佳。我们提出了探索性记忆增强同策略与异策略优化（EMPO$^2$），这是一种混合强化学习框架，它利用记忆进行探索，并结合了同策略和异策略更新，使大语言模型在有记忆的情况下表现良好，同时确保在没有记忆的情况下的鲁棒性。在 ScienceWorld 和 WebShop 上，EMPO$^2$ 分别比 GRPO 提高了 128.6% 和 11.3%。此外，在分布外测试中，EMPO$^2$ 展示了对新任务的卓越适应能力，仅需少量记忆尝试且无需参数更新。这些结果表明 EMPO$^2$ 是构建更具探索性和泛化能力的大语言模型智能体的一个极具前景的框架。

## Abstract
Exploration remains the key bottleneck for large language model agents trained with reinforcement learning. While prior methods exploit pretrained knowledge, they fail in environments requiring the discovery of novel states. We propose Exploratory Memory-Augmented On- and Off-Policy Optimization (EMPO$^2$), a hybrid RL framework that leverages memory for exploration and combines on- and off-policy updates to make LLMs perform well with memory while also ensuring robustness without it. On ScienceWorld and WebShop, EMPO$^2$ achieves 128.6% and 11.3% improvements over GRPO, respectively. Moreover, in out-of-distribution tests, EMPO$^2$ demonstrates superior adaptability to new tasks, requiring only a few trials with memory and no parameter updates. These results highlight EMPO$^2$ as a promising framework for building more exploratory and generalizable LLM-based agents.

### 2. 方法论：EMPO² 框架
EMPO² 结合了参数化学习（模型权重更新）和非参数化学习（外部记忆更新），其核心思想是通过“记忆引导探索”并利用“异策略优化内化知识”。

*   **自生成记忆（Self-Generated Memory）**：智能体在每次尝试失败后，利用 LLM 自身的能力对轨迹进行反思，生成“提示建议（Tips）”并存入外部记忆库。
*   **混合采样模式（Hybrid Rollout）**：
    *   **无记忆采样**：仅根据当前状态和任务生成动作。
    *   **记忆增强采样**：检索记忆库中最相关的 Tips 作为上下文，引导智能体避开之前的错误，探索新路径。
*   **混合优化模式（Hybrid Update）**：
    *   **同策略更新（On-Policy）**：在有 Tips 或无 Tips 的情况下，直接优化当前策略。
    *   **异策略更新（Off-Policy）**：这是关键创新。将“带 Tips 引导下生成的成功轨迹”作为教师，让“不带 Tips 的基础策略”去学习。通过重要性采样（Importance Sampling）和掩码机制（Masking），将 Tips 提供的外部引导转化为模型自身的内在参数能力。
*   **内在奖励（Intrinsic Reward）**：引入基于状态新颖性的奖励（类似于 Count-based exploration），鼓励智能体访问未曾见过的状态。
