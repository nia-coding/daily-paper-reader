## 摘要
多智能体辩论（MAD）系统利用集体智慧来增强推理能力，然而现有方法难以同时优化准确性、共识形成和计算效率。静态拓扑方法缺乏对任务复杂度变化的适应性，而基于外部大语言模型（LLM）的协调机制则存在引入特权知识的风险，从而损害辩论的中立性。本研究提出了 RUMAD（Reinforcement-Unifying Multi-Agent Debate），这是一个将 MAD 中的动态通信拓扑控制建模为强化学习（RL）问题的新型框架。RUMAD 采用了一种与内容无关的观测方案，通过捕捉高层辩论动态来避免访问智能体的原始推理内容。RUMAD 使用多目标奖励来建模解的质量、凝聚力和效率。一个经过 PPO 训练的控制器动态调整通信图中的边权重，而双阈值机制则实现了对智能体激活和信息可见性的细粒度控制。在 MMLU、GSM8K 和 GPQA 基准测试上的实验评估表明，与单个 LLM 模型和多个 MAD 基准相比，RUMAD 实现了显著的效率提升，将 Token 成本降低了 80% 以上，同时仍提高了推理准确性。值得注意的是，仅在 MMLU 上训练的 RUMAD 在域外（OOD）任务上表现出鲁棒的零样本泛化能力，这表明所学习的通信策略捕捉到了有效多智能体协调中与任务无关的原则。这些结果确立了 RUMAD 是一种在实际资源约束下部署多智能体推理应用的高效且鲁棒的方法。

## Abstract
Multi-agent debate (MAD) systems leverage collective intelligence to enhance reasoning capabilities, yet existing approaches struggle to simultaneously optimize accuracy, consensus formation, and computational efficiency. Static topology methods lack adaptability to task complexity variations, while external LLM-based coordination risks introducing privileged knowledge that compromises debate neutrality. This work presents RUMAD (Reinforcement-Unifying Multi-Agent Debate), a novel framework that formulates dynamic communication topology control in MAD as a reinforcement learning (RL) problem.   RUMAD employs a content-agnostic observation scheme that captures high-level debate dynamics avoiding access to raw agent reasoning content. RUMAD uses a multi-objective reward to model solution quality, cohesion and efficiency. A PPO-trained controller dynamically adjusts edge weights in the communication graph, while a dual-threshold mechanism enables fine-grained control over both agent activation and information visibility.   Experimental evaluation across MMLU, GSM8K, and GPQA benchmarks demonstrates that RUMAD achieves substantial efficiency gains, reducing token costs by over 80\%, while still improving reasoning accuracy compared to single LLM model and multiple MAD baselines. Notably, RUMAD trained exclusively on MMLU exhibits robust zero-shot generalization to out-of-domain (OOD) tasks, indicating that the learned communication strategies capture task-independent principles of effective multi-agent coordination. These results establish RUMAD as a efficient and robust approach for deploying multi-agent reasoning application with practical resource constraints.

### RUMAD：强化统一的多智能体辩论框架论文总结

#### 1. 核心问题与研究动机
多智能体辩论（MAD）通过多个大语言模型（LLM）的交互来提升推理能力，但面临三大挑战：
*   **适应性不足**：现有方法多采用静态拓扑（如环形、星形），无法根据任务难度动态调整通信开销。
*   **隐私与中立性风险**：引入外部 LLM（如 GPT-4）作为裁判会暴露辩论内容，且可能引入外部偏见，抑制智能体群体的多样性。
*   **效率瓶颈**：缺乏对 Token 成本的显式建模，导致在实际部署中资源浪费严重。
**研究动机**：开发一种既能保证辩论中立性，又能根据任务复杂度动态优化通信拓扑，从而在资源受限下实现高精度推理的框架。

#### 2. 方法论：RUMAD 框架
RUMAD 将动态拓扑控制建模为强化学习（RL）问题，核心组件包括：
*   **内容无关观测（Content-Agnostic Observation）**：控制器不读取辩论的原始文本，仅观测由语义相似度标量、答案一致性、辩论进度和通信成本构成的 $N \times N$ 矩阵。这确保了控制器的中立性。
*   **PPO 控制器**：使用近端策略优化（PPO）算法训练一个非 LLM 的轻量级控制器。它输出智能体间通信边的权重分布（基于高斯分布采样）。
*   **双阈值机制**：
    *   **激活阈值**：若某智能体受到的外部影响权重低于阈值，则该轮被“静默”（复用旧回复），从而节省 Token。
    *   **可见性阈值**：根据权重将邻居信息分为“关键”、“参考”、“背景”和“不可见”四个等级，精细化构建提示词。
*   **多目标奖励函数**：综合考虑解的质量（准确率、改进度）、群体凝聚力（共识度）和资源效率（Token 节省、稀疏度惩罚）。

#### 3. 实验设计
*   **数据集**：
    *   **训练集**：仅在 MMLU dev 集上训练。
    *   **测试集**：MMLU test（综合）、GSM8K（数学推理）、GPQA（研究生级科学问答）。
*   **Benchmark 与对比方法**：
    *   **基准**：全连接 MAD、S-MAD（星形/环形静态拓扑）、GD（分组辩论）、S2-MAD（两阶段稀疏辩论）。
*   **智能体配置**：由 6 个智能体组成（Llama-3.1-8B、ChatGLM-4-9B、Deepseek-Math-7B 各两个），采用异构架构以增加辩论多样性。

#### 4. 资源与算力
*   **硬件**：实验在一台 **NVIDIA GeForce RTX 3090 GPU** 上完成。
*   **优化技术**：所有 LLM 均使用 **4-bit 量化**（Q4_K_M）和混合精度部署，以支持在单卡上同时运行 6 个智能体。
*   **训练细节**：控制器训练过程轻量，使用了 Nomic-Embed-v1 模型计算语义相似度。

#### 5. 实验数量与充分性
*   **实验规模**：涵盖了三个主流基准测试，并进行了跨领域的 **Zero-shot（零样本）迁移实验**。
*   **消融实验**：针对奖励函数组件（$R_t$ 与 $R_{ep}$）、动作分布类型（高斯 vs Beta）、智能体激活机制、预算损失函数等进行了详尽测试。
*   **扩展性分析**：测试了不同智能体数量（3, 6, 9 个）和不同通信预算（B=6, 12, 18）下的表现。
*   **评价**：实验设计客观公平，通过 OOD（域外）任务验证了策略的通用性，而非仅仅在特定任务上过拟合。

#### 6. 主要结论与发现
*   **高效性**：RUMAD 在 MMLU 和 GSM8K 上比全连接 MAD **节省了 80% 以上的 Token 成本**。
*   **准确性**：在大幅减小开销的同时，RUMAD 的推理准确率优于所有静态拓扑基准（如在 MMLU 上从 49% 提升至 68%）。
*   **强泛化性**：仅在 MMLU 上训练的控制器，在完全不同的 GSM8K（数学）和 GPQA（科学）任务上依然表现卓越，证明其学到了通用的协作原则。
*   **自适应分配**：在固定预算下，增加智能体数量反而可能降低 Token 成本，因为控制器学会了更精准地挑选少数专家进行发言。

#### 7. 优点
*   **内容无关性**：保护了辩论的隐私和中立性，避免了“裁判”干预推理逻辑。
*   **极高的资源效率**：通过动态剪枝和智能体静默机制，极大地降低了 LLM 推理的经济成本。
*   **可解释的预算控制**：通过参数 $B$ 允许用户根据实际资源约束灵活调整辩论的激进程度。

#### 8. 不足与局限
*   **中心化瓶颈**：虽然 6-8 个智能体表现良好，但当智能体规模达到数百个时，中心化的 PPO 控制器可能面临计算压力。
*   **量化影响**：实验主要基于 4-bit 量化模型，虽然公平对比了基准，但在全精度模型上的表现差异有待进一步验证。
*   **任务类型限制**：目前主要针对选择题和简答题，对于长文本生成或多模态辩论的适用性尚未探讨。
