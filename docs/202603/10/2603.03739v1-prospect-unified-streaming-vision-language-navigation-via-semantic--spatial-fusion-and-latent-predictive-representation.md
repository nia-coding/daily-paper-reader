## 摘要
多模态大语言模型（MLLMs）推动了零样本端到端视觉语言导航（VLN）的发展，但鲁棒的导航不仅需要语义理解，还需要对环境动态和空间结构进行预测建模。我们提出了 PROSPECT，这是一个统一的流式导航智能体，它将流式视觉-语言-动作（VLA）策略与潜空间预测表示学习相结合。PROSPECT 使用 CUT3R 作为流式 3D 基础空间编码器，以生成长上下文、绝对尺度的空间特征，并通过交叉注意力机制将其与 SigLIP 语义特征融合。在训练过程中，我们引入了可学习的流查询令牌（stream query tokens），用于查询流式上下文并预测下一步的 2D 和 3D 潜空间特征（而非像素或显式模态），并在冻结的 SigLIP 和 CUT3R 教师模型的潜空间中进行监督。预测分支在不增加推理开销的情况下塑造了内部表示。在 VLN-CE 基准测试和真实机器人部署上的实验表明，该方法达到了最先进的性能，并在多种光照条件下提高了长时程鲁棒性。我们将很快向社区发布代码。

## Abstract
Multimodal large language models (MLLMs) have advanced zero-shot end-to-end Vision-Language Navigation (VLN), yet robust navigation requires not only semantic understanding but also predictive modeling of environment dynamics and spatial structure. We propose PROSPECT, a unified streaming navigation agent that couples a streaming Vision-Language-Action (VLA) policy with latent predictive representation learning. PROSPECT uses CUT3R as a streaming 3D foundation spatial encoder to produce long-context, absolute-scale spatial features, and fuses them with SigLIP semantic features via cross-attention. During training, we introduce learnable stream query tokens that query the streaming context and predict next-step 2D and 3D latent features (rather than pixels or explicit modalities), supervised in the latent spaces of frozen SigLIP and CUT3R teachers. The predictive branch shapes internal representations without inference overhead. Experiments on VLN-CE benchmarks and real-robot deployment demonstrate state-of-the-art performance and improved long-horizon robustness under diverse lighting. We will release code for the community soon.

### 1. 核心问题与整体含义（研究动机和背景）
视觉语言导航（VLN）要求智能体根据自然语言指令在未知环境中移动。虽然现有的多模态大模型（MLLM）在语义理解上表现出色，但仍面临两个核心挑战：
*   **缺乏预测能力：** 现有的视觉-语言-动作（VLA）模型大多只关注当前动作生成，缺乏对环境动态和空间结构的预测建模（即缺乏“世界模型”能力）。
*   **空间感知不足：** 许多模型依赖 2D 语义编码器，难以处理复杂的 3D 空间结构，且在处理长序列流式数据时容易出现内存溢出（OOM）或尺度不一致问题。
*   **监督噪声：** 现有的预测方法若在像素或深度空间进行监督，容易过拟合于纹理、光照等任务无关的细节。

### 2. 方法论：核心思想与关键技术
PROSPECT 提出了一个统一的流式导航框架，其核心思想是将**流式 VLA 策略**与**潜空间预测表示学习**相结合：
*   **双流感知与融合：**
    *   **2D 语义流：** 使用 SigLIP 提取图像语义特征。
    *   **3D 空间流：** 采用 **CUT3R** 作为流式 3D 基础编码器，提供具有“绝对尺度”的空间特征，适合长程流式处理。
    *   **融合机制：** 通过交叉注意力（Cross-Attention）将 2D 语义与 3D 空间特征融合，并映射到 LLM 的嵌入空间。
*   **潜空间预测（Latent Prediction）：**
    *   引入**流查询令牌（Stream Query Tokens）**：在训练阶段，向 LLM 输入追加可学习的 2D 和 3D 查询令牌。
    *   **预测目标：** 预测下一时刻（$t+1$）的 2D 语义和 3D 空间潜空间特征，而非原始像素。
    *   **监督信号：** 使用冻结的 SigLIP（余弦相似度损失）和 CUT3R（MSE 损失）作为教师模型进行监督。
*   **流式注意力掩码（Streaming Attention Mask）：**
    *   设计了特殊的掩码以确保：1) 时间因果性；2) 不同回合间的查询令牌相互隔离；3) 2D 与 3D 预测任务解耦，防止模态间的信息泄露。
*   **推理优化：** 预测分支仅在训练时存在，推理时将其移除，因此**不增加任何推理开销**，但模型内部表征已通过预测任务得到了强化。

*   **鲁棒性：** 模型在真实世界中表现出极强的光照鲁棒性，能够处理夜晚街道等极端低光照环境。

### 7. 优点
*   **高效性：** 预测分支在推理时移除，实现了“训练时强化，推理时零开销”。

*   **长上下文处理：** 结合流式 3D 编码器和长短期记忆机制，有效解决了长程导航中的记忆衰减问题。
*   **实机泛化：** 展现了极强的零样本迁移能力，无需在特定实机场景微调。

### 8. 不足与局限
*   **推理依赖：** 虽然模型本身高效，但由于使用了 7B 规模的 LLM，实机部署仍需依赖远程服务器或高性能移动 GPU（如 RTX 4070），在低功耗嵌入式设备上运行受限。
