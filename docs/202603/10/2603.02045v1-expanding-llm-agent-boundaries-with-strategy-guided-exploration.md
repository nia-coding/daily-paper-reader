## 摘要
强化学习（RL）在大语言模型（LLM）的后训练中取得了显著成功，使其能够作为智能体执行计算机使用、工具调用和编程等任务。然而，探索仍然是 LLM 智能体强化学习中的核心挑战，特别是当它们在具有复杂观测和稀疏结果奖励的语言-动作空间中运行时。在这项工作中，我们通过利用 LLM 在语言中对环境进行规划和推理的能力，将探索从低级动作转向高级语言策略，从而解决了 LLM 智能体的探索问题。因此，我们提出了策略引导探索（Strategy-Guided Exploration, SGE），它首先生成简洁的自然语言策略，描述为实现目标应采取的步骤，然后根据该策略生成环境动作。通过在策略空间而非动作空间进行探索，SGE 引导了针对不同环境结果的结构化且多样化的探索。为了在强化学习期间增加策略多样性，SGE 引入了混合温度采样（mixed-temperature sampling），以并行探索多种策略，并结合策略反思过程，根据环境中先前策略的结果来生成策略。在 UI 交互、工具调用、编程和具身智能体环境中，SGE 一贯优于专注于探索的强化学习基线，提高了学习效率和最终性能。我们展示了 SGE 使智能体能够学会解决对于基础模型而言过于困难的任务。

## Abstract
Reinforcement learning (RL) has demonstrated notable success in post-training large language models (LLMs) as agents for tasks such as computer use, tool calling, and coding. However, exploration remains a central challenge in RL for LLM agents, especially as they operate in language-action spaces with complex observations and sparse outcome rewards. In this work, we address exploration for LLM agents by leveraging the ability of LLMs to plan and reason in language about the environment to shift exploration from low-level actions to higher-level language strategies. We thus propose Strategy-Guided Exploration (SGE), which first generates a concise natural-language strategy that describes what to do to make progress toward the goal, and then generates environment actions conditioned on that strategy. By exploring in the space of strategies rather than the space of actions, SGE induces structured and diverse exploration that targets different environment outcomes. To increase strategy diversity during RL, SGE introduces mixed-temperature sampling, which explores diverse strategies in parallel, along with a strategy reflection process that grounds strategy generation on the outcomes of previous strategies in the environment. Across UI interaction, tool-calling, coding, and embodied agent environments, SGE consistently outperforms exploration-focused RL baselines, improving both learning efficiency and final performance. We show that SGE enables the agent to learn to solve tasks too difficult for the base model.

### 2. 方法论：策略引导探索 (SGE)
SGE 的核心思想是：**先让模型生成一段自然语言描述的“策略”，再根据该策略执行具体动作。**

其关键技术细节包括：
*   **策略提示 (Strategy Prompting)**：在 Prompt 中要求模型在执行动作前，先输出一段简洁的自然语言策略（Strategy），描述如何达成目标。
*   **混合温度采样 (Mixed-Temperature Sampling)**：
    *   对“策略”部分的 Token 使用**高采样温度**（如 1.2），以诱导生成多样化的解题思路。
    *   对后续的“动作”部分使用**低采样温度**（如 0.7），以确保动作执行的精确性和稳定性。
    *   这种设计避免了直接在动作空间高热采样导致的“无效噪声”（如 UI 点击位置的微小偏移）。
*   **策略反思 (Strategy Reflection)**：
    *   **负向反思**：如果之前的尝试失败，将失败的策略序列放入上下文，要求模型进行批判性分析并生成一个截然不同的新策略。
    *   **正向反思**：利用成功的历史策略引导模型探索更多类似的成功路径，提高学习效率。
*   **算法框架**：SGE 建立在 **GRPO**（Group Relative Policy Optimization）之上，通过并行采样多个策略并根据结果计算优势函数。

### 6. 主要结论与发现
*   **突破性能天花板**：SGE 能够解决基础模型即使尝试 2048 次（pass@2048）也无法解决的难题，而传统 RL 基线往往受限于基础模型的初始能力。
*   **学习效率更高**：在所有测试环境中，SGE 的收敛速度和最终成功率均显著优于 RND 和 EntropyAdv 等探索基线。
*   **泛化能力强**：在未见过的 App 或新场景中，经 SGE 训练的模型表现出更好的迁移能力。
*   **模型规模效应**：SGE 的效果随模型规模增大而增强，但在极小模型（如 600M）上提升有限，因为小模型缺乏基本的策略推理能力。

### 7. 优点
*   **创新性探索空间**：巧妙地将 LLM 的语言推理能力转化为 RL 的探索动力，避开了低级动作空间的搜索爆炸。
*   **即插即用**：作为一种采样和提示策略，它可以轻松集成到 PPO、GRPO 等任何在线 RL 算法中。
*   **结构化探索**：相比于随机噪声，语言策略引导的探索更符合人类解决问题的逻辑，产生的轨迹质量更高。

### 8. 不足与局限
*   **推理成本增加**：由于每一步动作前都要生成一段策略描述，这增加了推理延迟和 Token 消耗，在对实时性要求极高的场景下受限。
*   **依赖基础能力**：SGE 假设基础模型已经具备一定的“说人话”和“做计划”的能力。如果基础模型太弱，生成的策略可能是无效的幻觉，从而误导探索。
*   **长程任务挑战**：在步骤极多（如数百步）的任务中，维护完整的策略历史进行反思可能会导致上下文窗口过载。
