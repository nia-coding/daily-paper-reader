## 摘要
现有的对具有记忆能力的智能体的评估通常将记忆与行动孤立开来。一类基准测试通过测试对过去对话或文本的回忆来评估记忆，但未能捕捉到记忆如何用于指导未来的决策。另一类则侧重于智能体在不需要长期记忆的单会话任务中的表现。然而，在现实场景中，记忆与行动是紧密耦合的：智能体在与环境交互时获取记忆，并随后依靠这些记忆来解决未来的任务。为了捕捉这一设定，我们推出了 MemoryArena，这是一个统一的评估平台，用于在多会话的“记忆-智能体-环境”循环中对智能体记忆进行基准测试。该基准测试由人工设计的智能体任务组成，具有显式相互依赖的子任务，智能体必须通过将经验提炼为记忆，从早期的行动和反馈中学习，并随后利用该记忆指导后续行动以完成整体任务。MemoryArena 支持网页导航、偏好约束规划、渐进式信息搜索和顺序形式推理等领域的评估。研究表明，在 LoCoMo 等现有长上下文记忆基准测试中表现接近饱和的智能体，在我们的智能体设定中表现不佳，这揭示了当前对具有记忆能力的智能体评估存在的差距。

## Abstract
Existing evaluations of agents with memory typically assess memorization and action in isolation. One class of benchmarks evaluates memorization by testing recall of past conversations or text but fails to capture how memory is used to guide future decisions. Another class focuses on agents acting in single-session tasks without the need for long-term memory. However, in realistic settings, memorization and action are tightly coupled: agents acquire memory while interacting with the environment, and subsequently rely on that memory to solve future tasks. To capture this setting, we introduce MemoryArena, a unified evaluation gym for benchmarking agent memory in multi-session Memory-Agent-Environment loops. The benchmark consists of human-crafted agentic tasks with explicitly interdependent subtasks, where agents must learn from earlier actions and feedback by distilling experiences into memory, and subsequently use that memory to guide later actions to solve the overall task. MemoryArena supports evaluation across web navigation, preference-constrained planning, progressive information search, and sequential formal reasoning, and reveals that agents with near-saturated performance on existing long-context memory benchmarks like LoCoMo perform poorly in our agentic setting, exposing a gap in current evaluations for agents with memory.

## 1. 核心问题与研究动机
*   **研究背景**：大语言模型（LLM）智能体具备记忆（存储知识）和行动（环境交互）两项核心能力。
*   **核心问题**：现有的评估基准将两者割裂。一类基准（如 LoCoMo）仅测试静态长文本的回忆，缺乏动态决策；另一类（如 WebArena）侧重单次会话行动，忽略了跨会话的长期记忆需求。
*   **研究动机**：在现实场景中，记忆与行动是紧密耦合的。智能体需要在交互中获取记忆，并利用记忆指导未来的决策。论文旨在填补这一空白，提出一个模拟“记忆-智能体-环境”闭环的评估基准。

## 2. 方法论：MemoryArena 框架
*   **核心思想**：引入**记忆-智能体-环境（MAE）循环**。任务被分解为多个具有显式因果依赖的子任务（会话），智能体必须在会话间持久化存储关键信息。
*   **关键技术细节**：
    *   **多会话工作流**：任务序列为 $[s_1 \to s_2 \to \dots \to s_n]$。每个子任务在独立会话中执行，会话间的交互轨迹不直接互通。
    *   **记忆系统操作**：定义了两个抽象函数：
        1.  **检索 (Retrieve)**：在当前步骤根据历史和指令从记忆库 $M$ 中提取相关信息。
        2.  **更新 (Update)**：子任务完成后，将当前交互轨迹提炼并整合进 $M$。
    *   **记忆分级**：将记忆方法分为 **0D**（原始上下文）、**1D**（扁平化提炼记忆）和 **2D**（结构化图/树状记忆）。
*   **算法流程**：智能体在每一步行动前先检索记忆，行动后获得环境反馈；子任务结束时更新持久化记忆，供后续子任务调用。

## 3. 实验设计
*   **评估场景（四个领域）**：
    1.  **捆绑网页购物**：购买一系列兼容产品（如相机机身、镜头、支架），后续购买必须符合前序产品的技术规格。
    2.  **群组旅行规划**：多名成员依次加入旅行，后续成员的偏好（如“比 Rebecca 的酒店高两级评分”）依赖于前序成员的决策。
    3.  **渐进式网页搜索**：约束条件随会话逐步增加，最终答案需满足所有累积条件。
    4.  **顺序形式推理**：基于数学和物理科研论文，将复杂的证明分解为相互依赖的引理推导。

*   **客观性**：任务由人类专家（如数学/物理博士）手工构建，确保了因果链条的严谨性和答案的唯一性，避免了自动化生成可能带来的逻辑漏洞。

## 6. 主要结论与发现
*   **性能鸿沟**：在现有长文本记忆基准上表现接近饱和的模型，在 MemoryArena 中的成功率极低（部分任务 SR 接近 0），证明了现有评估的局限性。
*   **外部记忆的局限**：外部记忆系统并不总是优于长上下文拼接。原因在于“表示不匹配”（压缩信息不如原始文本直观）和“训练不匹配”（智能体未针对检索到的记忆进行专门优化）。
*   **信念漂移**：随着子任务深度增加，性能显著下降。这表明智能体在多会话中难以维持准确的状态估计（POMDP 状态跟踪失败）。
*   **延迟代价**：外部记忆系统显著增加了端到端执行的延迟，且复杂性（2D 记忆）并不一定带来成比例的性能提升。

## 7. 优点
*   **任务设计新颖**：首次强调了记忆与行动在多会话、因果依赖任务中的耦合评估。
*   **高难度与高区分度**：通过复杂的约束链条，有效暴露了当前最强模型在长期状态跟踪上的短板。
*   **跨领域覆盖**：从日常购物到高深科研推理，覆盖了广泛的智能体应用场景。

## 8. 不足与局限
*   **样本规模**：相比于全自动生成的基准，人工构建的任务规模（数百个）相对较小，可能存在特定领域的偏差。
*   **模型依赖性**：实验结果高度依赖于底层模型（如 GPT-5-mini）的指令遵循能力，有时难以区分是“记不住”还是“理解错”。
*   **评估成本**：由于任务步骤长且涉及多次 API 调用，运行完整基准测试的经济成本和时间成本较高。
