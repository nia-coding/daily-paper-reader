## 摘要
移动机器人通常在各种开放、动态的场景中进行长时间部署，包括仓库和制造设施等室内环境，以及农业和道路作业等室外环境。一个核心挑战是构建一个可扩展的长时程记忆系统，以支持针对不同粒度的开放式指令进行规划、检索和推理的代理式工作流，同时为导航生成精确且可执行的答案。我们提出了 STaR，这是一个代理式推理框架，它 (i) 构建了一个与任务无关的多模态长期记忆，在保留细粒度环境语义（物体属性、空间关系和动态事件）的同时，能够泛化到未见过的查询；(ii) 引入了一种基于信息瓶颈（Information Bottleneck）原理的可扩展任务条件检索算法，从长期记忆中提取出一组紧凑、非冗余且信息丰富的候选记忆，用于上下文推理。我们在 NaVQA（室内外混合校园场景）和 WH-VQA（一个使用 Isaac Sim 构建的、包含许多视觉相似物体的定制仓库基准测试，强调上下文推理）上对 STaR 进行了评估。在这两个数据集上，STaR 始终优于强基准模型，实现了更高的成功率和显著更低的空间误差。我们进一步在真实的 Husky 轮式机器人上部署了 STaR，涵盖室内和室外环境，展示了其鲁棒的长时程推理能力、可扩展性和实用价值。项目网站：https://trailab.github.io/STaR-website/

## Abstract
Mobile robots are often deployed over long durations in diverse open, dynamic scenes, including indoor setting such as warehouses and manufacturing facilities, and outdoor settings such as agricultural and roadway operations. A core challenge is to build a scalable long-horizon memory that supports an agentic workflow for planning, retrieval, and reasoning over open-ended instructions at variable granularity, while producing precise, actionable answers for navigation. We present STaR, an agentic reasoning framework that (i) constructs a task-agnostic, multimodal long-term memory that generalizes to unseen queries while preserving fine-grained environmental semantics (object attributes, spatial relations, and dynamic events), and (ii) introduces a Scalable Task Conditioned Retrieval algorithm based on the Information Bottleneck principle to extract from long-term memory a compact, non-redundant, information-rich set of candidate memories for contextual reasoning. We evaluate STaR on NaVQA (mixed indoor/outdoor campus scenes) and WH-VQA, a customized warehouse benchmark with many visually similar objects built with Isaac Sim, emphasizing contextual reasoning. Across the two datasets, STaR consistently outperforms strong baselines, achieving higher success rates and markedly lower spatial error. We further deploy STaR on a real Husky wheeled robot in both indoor and outdoor environments, demonstrating robust long horizon reasoning, scalability, and practical utility.     *   **冗余与幻觉**：简单的检索增强生成（RAG）会引入大量冗余信息，导致大模型（LLM）产生幻觉或推理效率低下。
    *   **任务适应性**：现有的任务驱动型地图（如 Clio）在任务变更时往往需要重新处理所有原始数据，缺乏可扩展性。

### 2. 方法论
STaR 框架由 **OmniMem 记忆构建**和**基于信息瓶颈（IB）的检索算法**两部分组成：

*   **OmniMem（多模态记忆）**：
    1.  **视频字幕记忆 (C)**：使用视觉语言模型（NVILA）对视频片段生成描述，存储在向量数据库中。
    2.  **几何基元记忆 (P)**：通过开放词汇感知栈（Grounding-DINO, TAP 等）构建 3D 点云基元，记录语义特征和检测时间戳。
    3.  **视觉关键帧记忆 (K)**：以低频率（如 1Hz）保存带有物体轮廓标注的原始图像，用于细粒度视觉验证。
*   **STaR 检索算法（核心创新）**：
    *   **任务条件子集提取**：根据查询 $Q$ 从向量数据库检索高召回率的字幕，并关联对应的 3D 基元。
    *   **基于信息瓶颈（IB）的聚类**：利用 IB 原理，在保证任务相关信息最大化的前提下，通过迭代合并相邻基元来压缩记忆。公式上通过最小化 Jensen-Shannon 散度来衡量合并损失，从而生成紧凑的任务相关聚类。
    *   **非冗余证据选择**：在每个聚类中只选择一个最具代表性的字幕，并根据需要调用 LLM 提取关键帧图像，形成最终的紧凑证据集 $R$。
*   **代理式工作流**：使用 MLLM（如 GPT-4o-mini）作为规划器，自主决定搜索策略、调用 API 并进行跨模态推理，输出空间坐标、时间或描述性答案。

    *   **ReMEmbR**：基于视频字幕的 RAG 框架。

*   **实验规模**：在 NaVQA 上测试了 210 个查询，在 WH-VQA 上测试了 100 个任务。
*   **充分性**：实验覆盖了短、中、长三种时程，并针对不同任务类型（空间、时间、文本、多模态）进行了细分评估。

### 6. 主要结论与发现
*   **性能领先**：STaR 在所有时程设置下均优于基准模型。在长时程任务中，其成功率显著高于 ReMEmbR，且空间误差降低了 2 到 4 倍。
*   **可扩展性**：随着记忆长度增加，STaR 的推理耗时保持稳定，而传统方法往往因冗余信息增加而导致性能急剧下降。
*   **精准度**：通过 3D 基元中心点定位而非机器人观察位定位，STaR 提供的导航目标点更加精确。
*   **多模态优势**：结合关键帧视觉记忆，系统能够回答字幕中未涵盖的细粒度问题（如“货架上还有几个空位”）。

### 7. 优点
*   **信息压缩高效**：引入信息瓶颈（IB）原理进行任务驱动的聚类，有效解决了长程记忆中的冗余问题。
*   **多模态融合**：巧妙结合了 3D 几何、语言描述和原始图像，兼顾了宏观语义和微观细节。
*   **无需预定义任务**：与 Clio 等方法不同，STaR 支持开放式查询，且无需在任务变更时重新处理整个地图。
*   **实机部署验证**：在真实 Husky 机器人上完成了闭环测试，证明了算法的实用性。

### 8. 不足与局限
*   **预探索假设**：系统假设环境已被预先探索（通过遥控或自主探索构建记忆），不适用于完全陌生的即时部署场景。
*   **依赖外部 API**：推理核心依赖于 GPT-4o 等闭源大模型 API，这在无网络环境或对实时性要求极高的场景下可能受限。
*   **存储开销**：虽然通过关键帧管理降低了压力，但长达数天的部署仍会产生巨大的图像存储需求（文中提到 35 分钟记忆约需 4.3 GB 关键帧数据）。
*   **动态环境处理**：虽然利用了 4DMOS 过滤动态物体，但对于环境中物体位置的频繁变动（如仓库中货物的频繁搬运），记忆更新机制的细节描述较少。
