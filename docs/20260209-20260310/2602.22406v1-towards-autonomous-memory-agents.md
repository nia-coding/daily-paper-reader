## 摘要
最近的记忆智能体通过将经验和对话历史提取到外部存储中来改进大语言模型（LLMs）。这实现了低开销的上下文组装和在线记忆更新，而无需昂贵的 LLM 训练。然而，现有的解决方案仍然是消极且被动的；记忆的增长受限于偶然获得的信息，而记忆智能体在面临不确定性时很少寻求外部输入。我们提出了自主记忆智能体，能够以最低成本主动获取、验证和整理知识。U-Mem 通过以下方式实现这一构想：(i) 一种成本感知的知识提取级联机制，从廉价的自我/教师信号逐步升级到工具验证的研究，并在必要时寻求专家反馈；(ii) 语义感知的汤普森采样（Thompson sampling），用于平衡记忆的探索与利用，并缓解冷启动偏差。在可验证和不可验证的基准测试中，U-Mem 一贯优于先前的记忆基准，并能超越基于强化学习（RL）的优化，在 HotpotQA (Qwen2.5-7B) 上提升了 14.6 分，在 AIME25 (Gemini-2.5-flash) 上提升了 7.33 分。

## Abstract
Recent memory agents improve LLMs by extracting experiences and conversation history into an external storage. This enables low-overhead context assembly and online memory update without expensive LLM training. However, existing solutions remain passive and reactive; memory growth is bounded by information that happens to be available, while memory agents seldom seek external inputs in uncertainties. We propose autonomous memory agents that actively acquire, validate, and curate knowledge at a minimum cost. U-Mem materializes this idea via (i) a cost-aware knowledge-extraction cascade that escalates from cheap self/teacher signals to tool-verified research and, only when needed, expert feedback, and (ii) semantic-aware Thompson sampling to balance exploration and exploitation over memories and mitigate cold-start bias. On both verifiable and non-verifiable benchmarks, U-Mem consistently beats prior memory baselines and can surpass RL-based optimization, improving HotpotQA (Qwen2.5-7B) by 14.6 points and AIME25 (Gemini-2.5-flash) by 7.33 points.

### 1. 核心问题与整体含义（研究动机和背景）
论文指出，现有的记忆智能体（Memory Agents）大多是**被动且反应式**的。它们仅从已有的对话历史或交互中提取信息，记忆的增长受限于偶然获得的数据。当智能体遇到不确定性或推理失败时，很少主动寻求外部输入。
**研究动机**：为了打破这种“被动”局限，作者提出了 **U-Mem** 框架，旨在构建**自主记忆智能体**。这种智能体能够主动决定何时需要额外信息、从何处获取（教师、工具或专家），并在成本约束下持续进化，实现非参数化的性能提升。

### 2. 方法论：核心思想与关键技术
U-Mem 的核心是一个“检索-推理-进化”的循环，包含两大关键技术：

*   **成本感知的知识提取级联（Cost-Aware Extraction Cascade）**：
    针对失败的推理轨迹，系统不会盲目反思，而是按成本从低到高逐级寻求正确参考：
    1.  **Level 1 (教师 LLM)**：查询更强的模型（如 Gemini-3-flash）。
    2.  **Level 2 (工具增强教师)**：若 Level 1 失败，则让教师模型配合代码解释器等工具。
    3.  **Level 3 (人类专家/模拟)**：作为最后手段，咨询最高权威（实验中用最强模型模拟）。
    通过对比失败轨迹与正确参考，提取“全局程序记忆”和“局部纠错记忆”。

*   **语义感知汤普森采样（SA-CTS）**：
    为了解决新记忆的“冷启动”问题（即新记忆因缺乏反馈而无法被检索），U-Mem 将记忆效用建模为高斯分布 $N(\mu, \sigma^2)$：
    *   **初始化**：利用语义空间中 $n$ 个近邻的统计数据初始化新记忆的均值和方差。
    *   **概率检索**：在检索时从分布中采样效用值，结合语义相似度进行排序。这允许具有高不确定性（大方差）的新记忆有机会被选中。
    *   **贝叶斯更新**：根据任务结果（优势奖励 $r_{adv}$）动态更新记忆的后验分布。

*   **流式记忆演化**：
    通过 LLM 担任“整理者”，对记忆库进行**追加（Append）、合并（Merge）或剪枝（Prune）**，确保记忆库紧凑且高效。

*   **实验规模**：涵盖了 4 个不同性质的基准数据集，跨越了开源和闭源、不同参数规模（7B 到 14B 及以上）的多个模型。

### 6. 主要结论与发现
*   **性能卓越**：U-Mem 在所有基准测试中均显著优于现有记忆框架。在 HotpotQA 上比 Qwen2.5-7B 提升了 14.6 分。
*   **媲美 RL**：在不更新模型参数的情况下，U-Mem 的表现达到甚至超过了昂贵的强化学习（GRPO）方法，且方差更小，表现更稳定。
*   **成本效益**：通过级联机制，U-Mem 仅在 23% 的情况下调用昂贵的专家模型，就达到了 99% 以上的全专家调用性能。
*   **扩展性**：随着经验池（训练流）的增大，U-Mem 的性能呈现出类似“缩放法则”的持续增长趋势。

### 7. 优点
*   **主动性创新**：改变了记忆智能体被动记录的范式，引入了主动寻求外部验证的机制。
*   **非参数化进化**：无需梯度下降，适用于闭源模型，且避免了灾难性遗忘。
*   **探索机制**：引入汤普森采样解决了记忆检索中的“富者愈富”偏差，对新知识更友好。
*   **落地性强**：成本感知设计使其在实际部署中比全量微调或频繁调用昂贵模型更具经济性。

### 8. 不足与局限
*   **依赖外部信号**：虽然级联降低了成本，但系统仍高度依赖“教师模型”或“专家模型”的质量，若外部信号存在系统性偏差，记忆可能被污染。
*   **检索延迟**：尽管比 RL 快，但相比 No Memory，增加了检索和多轮推理的推理延迟。
*   **任务分布敏感**：实验显示，任务的重复性和结构化程度（如数学）越高，U-Mem 收益越大；对于极度发散、无规律的任务，收益可能收窄。
*   **模拟专家的局限**：实验中使用 Gemini-3-pro 模拟人类专家，这在现实中可能无法完全替代真实人类的复杂反馈。
